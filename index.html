What is CSS?
CSS stands for cascading style sheet.
CSS is optional but it converts an off looking HTML page into a beautiful and responsive website.

                background-color:seagreen;
            }

DOM
DOM stands for document object model.
When a page is loaded, the browser creates a DOM of page which acts as tree of objects.
HTML id and attributes
When an HTML element is given an id,it serves as an unique identifier for that element
On the other hand when an HTML element is given to a class , it now belongs to that class. More than one element can belong to a single class but every element must have a unique id(if assigned).

    CSS selectors -A CSS selector is used to select an HTMl element for styling

Important Notes

  • We can group selectors like this
    • h1,h2,h3{
      color:blue;
    }
  • We can use element class as selectors like this
    • p.red{
      color:red;
    }
  • '*' can be used as an universal selector to select all elements
    • *{
      margin:0;
      padding:0;
    }
